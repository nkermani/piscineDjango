Ex01 : Django, framework web.

Qu'est-ce que Django ?

Django est un framework web open-source écrit en Python qui suit le modèle MVC. Il a été créé en 2003 et publié en open-source en 2005.

Historique

  • 2003 : Création initiale
  • 2005 : Publication en open-source (licence BSD)
  • 2008 : Django 1.0 lancé
  • 2024 : Django continue d'évoluer

Caractéristiques principales

  • ORM pour l'abstraction de la base de données
  • Moteur de templates puissant
  • Interface d'administration automatique
  • Système de sécurité robuste
