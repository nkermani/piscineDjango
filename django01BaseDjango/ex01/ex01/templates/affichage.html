Ex01 : Processus d'affichage d'une page statique.

Etapes du processus

1. Requete HTTP

L'utilisateur accede a une URL dans son navigateur. Le navigateur envoie une requete HTTP GET au serveur.

2. Routage de l'URL

Django reçoit la requete et consulte le fichier urls.py pour router vers la bonne vue.

3. Execution de la vue

Django appelle la fonction de vue qui effectue la logique metier.

4. Chargement du template

La vue utilise render() pour charger un template HTML.

5. Compilation du template

Le moteur de templates remplace les variables et tags Django par du contenu reel.

6. Creation de la reponse HTTP

La vue retourne un objet HttpResponse contenant le HTML genere.

7. Affichage dans le navigateur

Le navigateur reçoit le HTML et le rend a l'ecran.

Diagramme du flux

Client HTTP -> urls.py -> view() -> render(template, context) -> HttpResponse -> Navigateur
