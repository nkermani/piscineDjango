Ex01 : Moteur de template.

Le moteur de templates de Django

Le moteur de templates Django permet de generer du HTML dynamique en combinant du HTML statique avec des donnees Python.

1. Les blocs (Blocks)

Un bloc est une zone du template qui peut etre remplacee par les templates enfants pour respecter le principe DRY.

{% block nom %}contenu{% endblock %}

2. Les boucles (for ... in)

Les boucles permettent de parcourir une liste et de repeter du code HTML pour chaque element.

{% for item in items %}
  <p>{{ item }}</p>
{% endfor %}

3. Les structures de controle (if)

Les conditions permettent d'afficher du contenu de maniere conditionnelle.

{% if condition %}
  <p>Vrai</p>
{% else %}
  <p>Faux</p>
{% endif %}

4. Affichage de variables (contexte)

Les variables passees par contexte sont accessibles via {{ variable }}.

Exemple : la vue passe {"name": "Alice"} au template, accessible via {{ name }}.
