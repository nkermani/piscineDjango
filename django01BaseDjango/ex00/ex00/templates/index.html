Ex00 : Markdown Cheatsheet.

Headers

Markdown Syntax Result
# Header 1 Header level 1
## Header 2 Header level 2
### Header 3 Header level 3
#### Header 4 Header level 4
##### Header 5 Header level 5
###### Header 6 Header level 6

Emphasis

Markdown Syntax Result
*italic* or _italic_ italic
**bold** or __bold__ bold
***bold and italic*** bold and italic
~~strikethrough~~ strikethrough

Lists

Unordered Lists

Markdown Syntax Description
- Item 1 Dash syntax
* Item 1 Asterisk syntax
+ Item 1 Plus syntax

Ordered Lists

Markdown Syntax Description
1. First item Numbered list
2. Second item Continues numbering

Nested Lists

- Item 1
  - Nested item 1
  - Nested item 2
- Item 2

Links

Markdown Syntax Description
[Link text](URL) Inline link
[Link text](URL "Title") Link with title
[Link text][reference] Reference link
<URL> Automatic link

Images

Markdown Syntax Description
![Alt text](image_url) Inline image
![Alt text](image_url "Title") Image with title
![Alt text][reference] Reference image

Code

Markdown Syntax Description
`inline code` Inline code with backticks
```language
code block
```		 Fenced code block
4 spaces indentation Code block with indentation

Blockquotes

Markdown Syntax Description
> Quote text Simple blockquote
> Line 1
> Line 2		 Multi-line blockquote
> Level 1
>> Level 2		 Nested blockquote

Horizontal Rules

Markdown Syntax Description
--- Three dashes
*** Three asterisks
___ Three underscores

Tables

| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Cell 1   | Cell 2   | Cell 3   |
| Cell 4   | Cell 5   | Cell 6   |

Table Alignment

| Left     | Center   | Right    |
|:---------|:--------:|---------:|
| Left     | Center   | Right    |

Task Lists

Markdown Syntax Description
- [ ] Task Unchecked task
- [x] Task Checked task

Line Breaks

Markdown Syntax Description
Two spaces at end of line Soft line break
Empty line New paragraph
<br> HTML line break

Footnotes

Markdown Syntax Description
Text[^1] Footnote reference
